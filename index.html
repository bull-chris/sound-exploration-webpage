Piano

Sound compass is a musical instrument made using microbit. Microbit is a small circuit board with multiple sensors in it. for this instrument, we are using a compass sensor in-built in microbit. when the user rotates this board at a certain angle it produces piano sounds. there are 8 pre-defined sounds for 8 different angles.
